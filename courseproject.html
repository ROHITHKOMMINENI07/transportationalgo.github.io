Transportation Management System Features

1. Find Shortest Route (Dijkstra’s Algorithm):

  • Computes the shortest path between a specified source and destination using Dijkstra’s Algorithm.
  • Displays the shortest distance and the sequence of locations in the path.
  • If no path exists, it notifies the user.
  • Uses a priority queue (implemented with a set) to select the next closest node during the algorithm's execution.

2. Find Minimum Spanning Tree (Prim's Algorithm):

  • Uses Prim's Algorithm to calculate the Minimum Spanning Tree (MST) of the graph.
  • Outputs the edges included in the MST and their weights (distances).
  • Calculates the total weight of the MST, representing the minimum cost to connect all locations in the network.

3. Sort Distances (Heap Sort):

  • Takes a list of distances as input and sorts them in ascending order using Heap Sort.
  • Outputs the sorted distances.
  • Heap Sort is an efficient, comparison-based sorting algorithm that uses a binary heap data structure.

4. Search Substring in Text (Rabin-Karp Algorithm):

  • Implements the Rabin-Karp Algorithm to search for a substring (pattern) within a given text.
  • Outputs whether the pattern is found in the text.
  • Uses hashing to find any one of a set of pattern strings in the text, making it efficient for multiple pattern searches.
